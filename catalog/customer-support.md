# customer-support plugin

**Marketplace:** `knowledge-work-plugins/customer-support`
**Status in this repo:** cataloged only — not yet installed locally.

## Description

Triage tickets, draft responses, escalate issues, and build your knowledge base. Research customer context and turn resolved issues into self-service content.

## Install

```bash
claude plugins add knowledge-work-plugins/customer-support
```

## Skills

| Skill                   | When to use                                                                                |
| ----------------------- | ------------------------------------------------------------------------------------------ |
| `ticket-triage`         | Sort and prioritize incoming tickets — urgency, category, owner                            |
| `draft-response`        | Tailored customer-facing response for product questions, escalations, declines, billing    |
| `kb-article`            | Turn a resolved issue into a knowledge-base article                                        |
| `customer-escalation`   | Package an escalation for engineering, product, or leadership with full context           |
| `customer-research`     | Multi-source research on a customer question with source attribution                       |
